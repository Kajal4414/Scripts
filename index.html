ASUS Configuration

Battery threshold settings and ASUS service management.

Set Battery Charging Threshold (60%)

Requires the ASUS System Control Interface driver. Does not need the MyASUS app.

1

Command Prompt (Administrator)

cmd
reg add "HKLM\SOFTWARE\ASUS\ASUS System Control Interface\AsusOptimization\ASUS Keyboard Hotkeys" /v ChargingRate /t REG_DWORD /d 60 /f
2

Registry Editor (Manual)

Navigate to the following key, then set ChargingRate to 3C (hex) or 60 (decimal).

HKLM\SOFTWARE\ASUS\ASUS System Control Interface\AsusOptimization\ASUS Keyboard Hotkeys
3

PowerShell Script (Alternative)

Use the pre-built script at ../Extra/charging_threshold.ps1.

ASUS Services Management

powershell — list all asus services
Get-Service | Where-Object { $_.Name -like "*ASUS*" }

ASUS services (safe to disable all except ASUSOptimization):

AsusAppService ASUSLinkNear ASUSLinkRemote ASUSSoftwareManager ASUSSwitch ASUSSystemAnalysis ASUSSystemDiagnosis
powershell — disable non-essential services
Set-Service -Name "AsusAppService"       -StartupType Disabled
Set-Service -Name "ASUSLinkNear"         -StartupType Disabled
Set-Service -Name "ASUSLinkRemote"       -StartupType Disabled
Set-Service -Name "ASUSSoftwareManager"  -StartupType Disabled

Set-Service -Name "ASUSSystemAnalysis"   -StartupType Disabled
Set-Service -Name "ASUSSystemDiagnosis"  -StartupType Disabled
powershell — restart optimization service
Restart-Service "ASUSOptimization"

Disk & UEFI Management

Guides for managing partitions, fixing GRUB, and rebuilding the Windows BCD Store.

Removing Windows Recovery Partition

Open Command Prompt as Administrator and run the following commands in sequence:

cmd — diskpart
diskpart
list disk
select disk 0         # Replace '0' with the disk containing recovery partition
list partition
select partition X    # Replace 'X' with the recovery partition number
delete partition override
exit

Removing Linux from GRUB Boot Menu

For dual-boot setups. Open Command Prompt as Administrator.

cmd — assign efi letter
diskpart
list disk
select disk 0         # Replace '0' with your system disk
list partition
select partition 0    # Replace '0' with your EFI System Partition number
assign letter=R
exit
cmd — remove boot entry
R:
cd EFI
dir                   # Verify directory contents
rd ubuntu /s          # Replace 'ubuntu' with the correct Linux folder name
# Type 'Y' and press Enter to confirm deletion. Finally, Reboot.

Backup & Restore UEFI Partition (BCD Store)

Backup your Boot Configuration Data before making sensitive changes.

cmd — backup
bcdedit /export "C:\BCD_Backup.bcd"
cmd — restore
bcdedit /import "C:\BCD_Backup.bcd"

Format & Recreate UEFI Partition

If the UEFI partition is corrupted, you can reformat and rebuild it from Command Prompt (Admin) or Windows Setup (Shift + F10).

cmd — format existing efi
diskpart
list disk
select disk 0         # Disk containing the UEFI partition
list partition        # Look for the 100MB-1GB FAT32 EFI partition
select partition 1    # Replace '1' with your EFI partition number
format fs=fat32 quick label="System"
assign letter=V
exit
If the partition is completely missing, replace the format line above with:
create partition efi size=260
cmd — rebuild bcd & cleanup
bcdboot C:\Windows /s V: /f UEFI

diskpart
select volume V
remove letter=V
exit
